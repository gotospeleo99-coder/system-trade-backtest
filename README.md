# system-trade-backtest

## 概要

本プロジェクトは開発中で、日本株を対象としたスイングトレード向けバックテスト基盤です。
日足データを用い、数日〜数週間の保有期間を想定した戦略検証を目的としています。

本プロジェクトは学習目的および個人的なトレード戦略検証のために開発されています。

## 前提条件

本バックテストは以下の前提条件に基づいて設計されています。

* 日足データを使用
* スイングトレードを想定（数日〜数週間の保有）
* 売買シグナルは終値ベースで判定
* 約定は翌営業日の始値を想定

## 設計方針

本プロジェクトでは、シンプルで拡張しやすいバックテスト基盤を目指しています。

* 銘柄単体のシグナルを中心としたバックテスト
* 横断的なランキング・ポートフォリオ最適化は現時点では対象外
* 学習目的のためシンプルな設計を採用
* 将来的な機能拡張を考慮した構造設計

## 想定用途

* スイングトレード戦略の検証
* テクニカル指標のバックテスト
* 個人的なトレード戦略の研究
